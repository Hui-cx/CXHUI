## API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| name | 图标名称 | `string`  | `-` |
| color | 图标颜色| `string`  | `-` |
| width | 图标宽度 | `string`  | `1em` |
| height | 图标高度 | `string`  | `1em` |
| spin | 是否旋转 | `boolean`  | `flase` |
