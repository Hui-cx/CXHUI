# Breadcrumb 面包屑

显示当前页面在系统层级结构中的位置，并能向上返回

## API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| items | 面包屑内容 | `items[]` | `-` |
| separator | 分隔符符号 | `string` | ` / ` |
